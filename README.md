
# Personal Website

My personal website for introducing myself and presenting my work.


## Tech Stack

**Client:** Next, Typescript, Graphql, Locomotive Scroll

**Server:** Graphql API (Highgraph)


## Features

- Animated Scroll
- Graphql API
- Responsive
